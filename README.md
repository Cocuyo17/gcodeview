GCodeView
=========

Purpose
-------

A *FAST* gcode viewer.

Nothing fancy, nothing else.

Usage
-----

`gcodeview <file.gcode>`

Controls
--------

* __Scroll up/down__
	
	zoom in and out

* __PgUp / PgDn / shift + Scroll__
	
	go up and down layers

* __left click + drag__
	
	pan view

* __r__
	
	reset viewing window

* __q / ESC__
	
	exit


Dependencies
------------

1. SDL
2. OpenGL
3. FTGL
4. fontconfig
